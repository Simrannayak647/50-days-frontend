# Currency Converter

A clean and lightweight currency converter built using HTML, CSS, and JavaScript.  
It fetches real-time exchange rates and instantly updates the converted amount.

## Features
- Live currency conversion  
- Simple and intuitive interface  
- Supports multiple global currencies  

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- ExchangeRate-API  

## Usage
1. Open `index.html` in your browser  
2. Choose the currencies  
3. Enter the amount to convert  

## Project Files
- `index.html` — structure  
- `style.css` — design  
- `script.js` — logic
